<h2>六大话题专场</h2>
<p>上一届 D2, 我们邀请了 21 位讲师，为大家带来了涵盖 5 个专场的 20 个精美的话题，并且把它们都收录到了官方 Github 上，供大家学习与分享。</p>
<p>第十五届 D2 嘉宾集合</p>
<p>今年，本着「精益求精，体验创心」的理念，我们将专题拓展到了 6 场，更加强调前端「体验与效率」提升，希望把海内外优秀的话题都带给大家。一起看看今年都有哪些专场吧！</p>
<h3>语言与框架</h3>
<p>2020 年的 D2 语言与框架专场，我们邀请了 Ben 讲解如何快速重构和重写 RxJS 代码，Ujjwal 讲标准化 TC39 组织如何运作。
近年随着浏览器功能日益强大和 JS/TS 语言日渐丰富，前端生态也蓬勃发展，"前端 er"在研发链路中的地位也越来越重要。前端智能化应用，结合浏览器的 IoT 接口，天马行空，让人脑洞大开； React 的 useState 概念也火出了圈，让函数式编程快速进入了很多"前端 er"的工具箱，前端测试覆盖率 100% 不是梦；还有干掉让人不安的’any’ types 的新的 JS 编译语言 rescript…
今年的语言与框架专场，我们将邀请目前很火的 without ‘any’ types 的 Rescript 作者张宏波、tensorflow.js 的开发者 ping、以及圈内的众多大牛，来和大家分享生机勃勃的前端生态圈。</p>
<h3>跨端技术</h3>
<p>跨端技术的发展持续如火如荼，这背后反射的是消费互联网业务难以逃脱流量的逻辑，同时小程序又另辟蹊径，如此大前端开发者的面对挑战也愈发困难， 此时的我们不禁感慨“敢问路在何方”， 如何才能跳出“跨端不可能三角”，在“效率、质量、多端”三角下找到属于自己的答案？本届 D2 跨端技术专场将邀请该领域的技术专家，讨论跨端技术的未来，重新审视 Web 技术的发展，为各位带来有关跨端研发解决方案的新思路。</p>
<p>###前端体验
前端作为人机交互的实现者，也是离用户最近的研发，体验一直是我们不可或缺的责任。虽然前端的各类技术不断发展，但是对于体验的极致追求一直是我们的初心，从性能、稳定性、流畅性到交互效率、满意度等等，体验既是细节能力、又是系统工程，如何给予用户最优的产品体验是前端共同面临的基本命题，也是技术所能给予用户最直接的价值体现。
本届 D2 前端体验专场，让我们聚焦体验，回归初心，结合具体业务实践、技术探索，和一线技术专家一起，共同交流前端在体验方向的发展及经验。</p>
<h3>低代码</h3>
<p>2014 年，研究机构 Forrester Research 正式提出了“低代码”概念。进入 2021 年，随着互联网人口红利消失、国内云市场的竞争加剧，以及突如其来的新冠疫情，企业数字化转型浪潮再次加速，而这其中涉及到大量企业级中后台系统建设，中后台技术领域正在经历一场低代码（LowCode/LCDP/LCAP）主导的研发效率革命。
一方面，低代码研发模式正在变得越来越流行，另一方面，我们也看到低代码并不是万能的，关乎低代码技术的研究也逐步进入深水区，诸如自然布局、逻辑编排、模型驱动、智能向导、混合研发…
本专场将邀约当前活跃在低代码领域的一线技术专家，共同探讨低代码研发模式当前的发展、创新、变革和未来。</p>
<h3>Node/Serverless</h3>
<p>Serverless 作为一种新型的互联网架构直接或间接推动了云计算的发展，随着 Serverless 体系不断完善，也让整个前端开发的边界变的更大，未来的前端开发人员的视角会更广阔，不管是当前的全栈，还是边缘计算，AI，3D 等，前端会在不同领域去使用 Node.js 和 Serverless 结合，这将使得整个行业行业带来怎样的变革？我们如何基于自身的业务场景去运用，发挥 Serverless 的最大价值？
本专场将邀请一线技术专家分享相关技术实践，给大家带来更多的思考和经验参考。</p>
<h3>多样化</h3>
<p>在互联网行业的传统观念里，前端的主要任务是以页面的形式为各类网络服务构建用户交互界面。近年来，伴随着语言框架与跨端技术的发展，围绕着体验交互的核心命题，前端的边界在不断拓宽。技术的适用范围，从一般的互联网服务，延伸到诸如办公协作、工业制造、游戏 &amp; 多媒体等本地化应用的领域；应用的执行环境，从浏览器，扩展到 Web 容器，原生应用，IoT 设备；人机交互模式，从确定性的脚本，逐渐混合了人工智能的元素。
多样化的场景，开拓着我们的视野，同时，也在不断淬炼着我们对前端本质的认知。本届 D2 前端多样化专场，将继续秉持“技术无界”的理念，邀请方向与风格五彩斑斓的技术专家，为大家呈现前端的无限可能，探索潜藏其中的无限价值。</p>
